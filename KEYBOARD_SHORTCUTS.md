## üîß Troubleshooting

### Shortcuts Not Working?
1. Check if typing in input field
2. Close any open dialogs
3. Refresh page (F5)
4. Check browser console for errors

### Wrong Key Combination?
- Windows/Linux: Use `Ctrl`
- Mac: Use `Cmd` (‚åò)
- Check keyboard layout

### Conflicts with Browser?
Some shortcuts may conflict:
- `Ctrl + S` - Browser save (we override)
- `Ctrl + R` - Browser refresh (we override)
- Use `Ctrl + K` to see all available
